Commands syntax

· Commands all begin with a prefix.
The prefix is a little character, indicating a command. By default, Blade Bot prefix is - but it can be customized.

· Some commands have arguments.
An argument is something you put right next to the command name. Make sure to space each arguments!

· Which kind of arguments?
When you execute a command without its arguments, you might see a help-menu to show you the correct syntax. Between () are optional arguments, you're not obliged to mention them, between <> are must-have arguments, you're obliged to REPLACE the argument by its real content. (For exemple, you must replace "name" by a real username) Other arguments, without any delimitations, are must-have as well, and you must keep the same spelling and write the argument exactly as it is displayed in the help menu. Do NOT keep delimiters when you execute the command!
