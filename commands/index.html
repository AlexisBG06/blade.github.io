Commands syntax

· Commands all begin with a prefix.
The prefix is a little character, indicating a command. By default, Blade Bot prefix is - but it can be customized, look in customizable behaviour section!

· Some commands have arguments.
An argument is something you put right next to the command name. They may be variable according to the command. Make sure to replace arguments by their real content, and space them!

· which kind of argument?
When you execute a command without its arguments, you might see a help-menu to show you the arguments to place. Between () are optional arguments; you're not obliged to put them. Between <>, you must include the argument in your command, or it will still send the command help-menu. Between [] are the only possible words for this argument. Each possibilities is splitted with the | symbol.
